Kode di atas berfungsi untuk melakukan analisis dasar dataset menggunakan pustaka pandas dan matplotlib, dimulai dengan memuat file CSV ke dalam DataFrame, kemudian menampilkan 7 baris pertama serta informasi struktur dataset seperti jumlah baris, tipe data, dan kolom yang tersedia untuk memahami karakteristik data. Selanjutnya, kolom 'Dues amount' dibersihkan dengan menghapus karakter non-angka dan dikonversi menjadi tipe numerik agar bisa dihitung secara matematis. Setelah itu, program menghitung total kontribusi seluruh anggota serta rata-rata kontribusinya sebagai bentuk analisis statistik deskriptif. Terakhir, dibuat dua visualisasi yaitu histogram untuk melihat distribusi kontribusi dan diagram batang untuk membandingkan kontribusi masing-masing anggota berdasarkan Member ID, sehingga pola penyebaran dan perbandingan nilai dapat terlihat secara visual.
